Naive Set Theory

Paul Halmos

Career: All

Review: I was advised to read this book the summer before beginning graduate school. I found it to be an excellent introduction to the subject without lacking the necessary rigor. It's a necessary read for all aspiring mathematicians.

Linear Algebra

Friedberg, Insel, Spence

Career: All

Review: This is most likely the least known book on my list. The first half is ideal for undergraduates, while the second half includes some fairly advanced topics. I use it mainly as a reference for basic definitions and examples.
